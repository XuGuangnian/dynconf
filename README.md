## Dynconf

Dynamic configuration management for Rust.

## Features

- **Settings management** — Default values, validation, parsing, and templating.
- **Protection of sensitive information** — Passwords, tokens, and secrets (e.g. optional HashiCorp Vault integration).
- **Multiple file formats** — TOML, YAML, JSON, INI, and customizable loaders.
- **Environment variable overrides** — Full support for env overrides with optional dotenv support.
- **Optional layered environments** — Multi-environment layering (e.g. default, development, testing, production).

## Acknowledge

Design and feature set are inspired by [dynaconf](https://www.dynaconf.com/) — the dynamic configuration manager for Python.
